open-travel-planner
===================

Open Travel Planner

开放的旅行计划工具



公共交通规划工具
-------------------

1. 开放的航班数据库
2. 开放的火车时刻数据库
3. 开放的长途汽车数据库 (TODO) 

这个两个数据库除了用于路线计算。
还有一个很重要的用途是历史数据存档。
比如将来研究随着时间推移铁路网/航线的变化。
